# SiteMonitoria

#### Este projeto surgiu de um desafio proposto por um professor, que tem por finalidade ser a execução de um projeto próprio utilizando as tecnologias aprendidas no curso de análise e desenvolvimento de sistemas da universidade SENAI Joinville.

## Finalidade

#### Atualmente os registros das monitorias feitas na instituição são registradas em uma tabela Google Sheets, a criação do site vem para realizar a substituição dessa tabela, tornando mais prático o registro, podendo futuramente serem criadas novas funcionalidades para personalizar esses registros conforme a necessidade

## Tecnologias Utilizadas

- Java JDK 1.8
- Java JRE 17
- TomCat 9
- MySQL
- JQuery
- Bootstrap
